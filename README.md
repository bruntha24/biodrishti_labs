📘 BioDrishti — Life Sciences Research Mentorship Platform

BioDrishti is a modern, one-page web platform designed for structured research mentorship in Genomics, Molecular Biology, Immunology, Cancer Biology, Biomaterials, Bioinformatics, and related life science domains.

It serves as the official digital presence for students, institutions, and researchers to understand and connect with BioDrishti’s mentorship programs.

🚀 Live Purpose

This website acts as the front door of BioDrishti, helping:

Undergraduate students explore research pathways
Master’s students refine research direction
Early PhD students strengthen publication readiness
Institutions access workshops and advisory programs

⚙️ Getting Started
1. Clone repository
git clone https://github.com/bruntha24/biodrishti_labs.git
cd biodrishti_labs
2. Install dependencies
npm install
3. Run development server
npm run dev

App will run at:

http://localhost:5000
🏗 Build for Production
npm run build

Preview build:

npm run preview
🌐 Deployment

Recommended deployment:

Netlify (static hosting)
Vercel (alternative)

After build:

dist/ → deploy folder
